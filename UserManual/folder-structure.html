Estructura de Carpetas

Estrutura de Carpetas

Vamos percorrer os ficheiros e carpetas da app.

No raíz do proxecto temos os seguintes ficheiros e carpetas importantes.

1. /node_modules

Os paquetes npm instalados no proxecto co comando <npm install>.

2. /src

Cando navegamos na carpeta Src, podemos ver 4 outras carpetas dentro dela. Son a carpeta app, a carpeta assets, a carpeta pages e a carpeta theme.

Debaixo desta carpeta atoparás as seguintes carpetas e ficheiros importantes.

3. /assets

A carpeta Assets contén todas as imaxes, fontes que se usan na app. Tamén aquí se colocan outros activos da app.

4. /environments

Debaixo desta carpeta atópanse os ficheiros de configuración utilizados polo Angular CLI para xestionar as diferentes variables de entorno.

5. /theme

Contén todos os estilos, variables e mixins de Sass para seren utilizados na nosa app ionic.

6. /src/index.html

src/index.html é o punto de entrada principal da app, aínda que o seu propósito é configurar scripts, inclusións de CSS e bootstrap, ou comezar a executar a app. Non dedicaremos moito tempo a este ficheiro.

Comecemos coa carpeta app

A carpeta App é a máis grande porque contén todo o código da nosa app Angular. Ten todos os compoñentes, módulos, páxinas, servizos e estilos que usarás para construír a túa app.

Dentro da carpeta /app tamén hai algúns ficheiros importantes:

app.component.html

Isto serve como o esqueleto da app. Normalmente ten un para renderizar as rutas e o seu contido. Tamén pode estar envolto de contido que queiras que apareza en todas as páxinas.

app.component.ts

Define a estrutura básica e a navegación inicial da app.

app.routes.ts

Aquí definimos as rutas principais. As rutas secundarias doutras cargas preguizosas defínense dentro deses módulos. Estas rutas rexístranse no Angular RouterModule.

Carpeta Component

Esta carpeta contén os compoñentes que se usan en moitas páxinas en toda a app.

Carpeta Pages

Cada páxina ten a súa propia carpeta. Dentro desta carpeta atópanse todos os ficheiros relacionados coa páxina. Isto inclúe o HTML para o deseño, o Sass para os estilos e o compoñente (.ts) principal da páxina.

Carpeta Service

Esta carpeta é para todos os servizos que se empregan para acceder aos datos que se presentarán na app.

carpeta Tabs

Esta carpeta contén o componente de navegación de nivel superior para implementar unha navegación baseada en pestañas. O componente é un contenedor de compoñentes individuais de pestanas.

Páxinas

Vamos percorrer os ficheiros e carpetas das Páxinas.

Cada páxina ten a súa propia carpeta. Dentro desta carpeta atópanse todos os ficheiros relacionados coa páxina. Isto inclúe o HTML para o deseño, o Sass para os estilos e o compoñente principal da páxina.

Rutas

Atoparás todas as rutas principais en app/app.routes.ts e as rutas das pestañas en app/tabs/tabs-routing.module.ts

Compoñentes

Esta carpeta contén os compoñentes que se usan en moitas páxinas en toda a app.

Configuración de Idioma

Este proxecto admite varios idiomas: inglés e español, poderíamos ter todos os idiomas se traducimos os .json

Atoparás os ficheiros JSON de tradución na carpeta /assets/i18n/.

Para obter máis detalles sobre o módulo de traducións, consulta a documentación oficial de GitHub - https://github.com/ngx-translate/core
