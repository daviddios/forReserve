Introdución

O primeiro de todo é que estas diante dunha app feita con Ionic Ionic
Precisas de ter instalado no teu equipo, node e Angular CLI

Deixo por aquí o enlace á sua documentación

Preparación

Unha vez instalado node, instalaremos Angular CLI con npm:

npm install -g @angular/cli

Se todavia non instalaches Node, aquí vos deixo o enlace https://nodejs.org/es

Instalación

Logo de ter todo o anterior instalado, temos que clonar o repositorio,

https://github.com/daviddios/forReserve.git

Accedemos dentro do directorio que acabamos de clonar e instalamos as dependencias con

npm install

npm run start
